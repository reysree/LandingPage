## The Problem
Tracking personal finances across multiple bank accounts is historically fragmented and manual. Most solutions require giving up data privacy to third-party aggregators or manually entering data into spreadsheets.

## The Solution
An AI-powered finance application built using Next.js and Gemini AI. It allows users to track their finances securely, get intelligent insights about their spending habits, and interact with their financial data using natural language.

### Key Features
- **Real-time Synchronization**: Unified dashboard for multiple accounts
- **AI Financial Analyst**: Gemini-powered insights summarizing spending patterns
- **Natural Language Queries**: "How much did I spend on dining last month?"
- **Secure Architecture**: Data privacy-first design

<hr />

## Architecture & Stack

### Frontend
Built with **Next.js 15 App Router** and **React 19** for performant server-side rendering and efficient client components. **Tailwind CSS v4** combined with **shadcn/ui** provides a polished, minimal interface.

### Backend & AI Integration
The core value prop is powered by the Google **Gemini AI API**, utilizing structured prompting to analyze financial data payloads securely without storing sensitive PII in the model history. Database operations are handled via **PostgreSQL**.

<hr />

## Outcomes

- **Scalability**: Capable of processing thousands of historical transactions in seconds via efficient batching.
- **Accuracy**: 95%+ accuracy in transaction categorization using custom LLM prompts.
- **User Experience**: Drastically reduced the time users spend reconciling accounts from hours per month to minutes.
